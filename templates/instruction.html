📘 Instructions to Use

This project is a Python-based application designed to compare .csv and .xlsx files from two directories or zip archives. It detects differences in sheet names, headers, keys (strings), and numeric values, and generates a detailed summary report.

Problem statement:

This project solves that by providing a robust and automated comparison tool. It allows users to upload .csv or xlsx files or zip files containing spreadsheet data and performs an intelligent comparison by:

  • Matching files and sheets by name
  • Detecting differences in headers, string keys, and numeric values
  • Handling unordered rows through key-based logic
  • Providing a structured and visual report of discrepancies

The tool is especially useful for ETL testers, QA analysts, and data engineers who regularly verify data integrity between systems or after transformations.

🧩 Features:

  • ✅ Supports .csv and .xlsx file formats
  • ✅ Compares files and sheets by name
  • ✅ Performs row-by-row comparisons (ignores row order)
  • ✅ Detects mismatched keys and numerical differences
  • ✅ Generates a complete Excel summary report
  • ✅ Uses a modular architecture (reader, comparator, reporter)

📂 Input Requirements:

  • Only files with extensions .xlsx or .csv will be compared
  • Files should have the same file names, sheet names
  • All other file types will be ignored
  • You may upload a ZIP archive containing files
  • ZIP folder cannot contains another ZIP or folders - will be ignored
  • The application will scan and extract comparable files and generate a comparison report

📊 Output:

  • A result summary table displayed after comparison
  • An Excel file containing a highlighted report of all detected differences
